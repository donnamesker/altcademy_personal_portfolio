Video Production Database for Image Garden Inc.

Website Development & Database Design

The video production industry requires that estimates have a specific structure with line items listed in an above the line/below the line order. This site allows the user to create estimates, workorders, crew lists and call sheets for projects. It also allows the user to assign crew members and suppliers to each line item, making project coordination a breeze.

HTML, CSS, Bootstrap 5.2, Javascript, MySQL, PHP-PDO

This was a Shopify e-commerce website for our second company, Jokerbox.com. In 2010 my husband, Ben, invented a product designed for production professionals called a Jokerbox. Unfortunately, the pandemic forced us to close that business, so the website is no longer active.

A template was used for this site with a lot of cusomization. Our site was featured on the Shopify template custimaztion page for a short time. As you may know, Shopify uses the Liquid template language that is written in Ruby.

It was May of 1996...

We had just moved to Orlando, Florida from Cincinnati, Ohio and needed a website for our business. Donna taught herself HTML and CSS so she could create a site where clients could to see our work and check our availability. Donna has continued to learn about web development over the years and created various websites for our company, Image Garden Inc. She has customized websites from templates on WordPress and Shopify using the Liquid templating language written in Ruby. She has also created an estimating app for our video production company utilizing mySQL and PHP.
